# 디자인 시안 (클로드 디자인)

여기에 클로드 디자인으로 만든 화면 시안(HTML)을 넣는다. **참고용 시각 기준**이며 빌드에 포함되지 않는다.

## 넣을 파일
클로드 디자인 프로젝트의 10개 HTML 페이지 + `support.js`:

- 로그인 랜딩.dc.html
- 내 여행 목록.dc.html
- 플랜 뷰.dc.html
- 일정표.dc.html
- 장소.dc.html
- 예산 정산.dc.html
- 새 여행 만들기.dc.html
- 오버레이 3종.dc.html
- 뷰어 공유.dc.html
- 계정 설정.dc.html
- support.js

## 사용 원칙
- 구현 시 **시각 기준**으로만 참고한다. 코드를 그대로 복붙하지 않고 기능명세서·기획문서·설계문서를 따른다.
- 디자인 토큰(색·간격·radius·그림자)과 컴포넌트 스타일은 여기서 추출해 Tailwind/shadcn 설정에 반영한다.
